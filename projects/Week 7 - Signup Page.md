# Signup Page

Design the coolest signup page. 

## User Stories

-   [ ] Beautiful and intuitive design.
-   [ ] User can input name, email and password.
-   [ ] User can repeat password and see if it matches.

## Bonus features

-   [ ] Validation.
-   [ ] Redirect to a success page.
-   [ ] Animations.
